# nodejs_typescript_api

<h3> How Start This Project? </h3>

```
  First, Install Dependencies, Node, Yarn
```

<hr/>

```
  Run in terminal: yarn install
```

<hr/>

```
  After build the node_modules, run express server
```

```
  yarn dev
```

```
  use Postman, Insomnia to test API routes.
```
